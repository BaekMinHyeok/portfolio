01.Main Page
메인페이지는 저의 마인드맵으로 도출해낸 키워드인 ‘차분한, 평화로운’ 단어로 ‘잔잔한 파도’라는 컨셉을 구상하였습니다.
평소에 잔잔한 바다를 좋아해서 ‘calm sea’라는 문구가 앞으로의 방향성과 목표에 적합하다고 생각했습니다.
배경색은 다크그레이 기반으로 통일하여 디자인했고 메뉴는 후버시 물방울처럼 원으로 따라다니도록 표현했습니다.
메인 이미지는 로고의 물방울 색감과 잔잔한 파도 느낌을 나타내고 싶어서 이미지를 표현했습니다.
02.Branding
Branding 페이지의 메인은 3D 로고 이미지로 로고 제작 과정과 메인 컬러를 구체적으로 표현하기 위해 디자인했습니다.
전체 폰트는 Robot와 Noto Sans KR 기반으로 제목으로는 Noto Serif Display 폰트를 사용해서 차별성을 두었습니다.
03.About
About 메인 페이지는 액자 형태로 디자인했고 마인드맵과 경험, 앞으로의 목표를 간결하게 정리해서 제작했고 트렌지션효과와 후버시 스케일 효과로 시각적으로 간결하게 디자인했습니다.
